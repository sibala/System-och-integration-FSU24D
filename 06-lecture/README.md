### PRODUCTS endpoints

  - [GET]    http://localhost:3000/products
---

>Request JSON Body:
`None`

>Response JSON Body:
``` 
[
  {
    "id": 9,
    "name": "Product name",
    "description": "Desc...",
    "price": 423,
    "stock": 234,
    "category": "Graduation Cake",
    "image": "image url",
    "created_at": "2025-03-11T13:59:05.000Z"
  }
]
``` 


  - [GET]    http://localhost:3000/products/:id
---
>Request JSON Body: 
`None`

>Response JSON Body:
``` 
{
  "id": 9,
  "name": "Product name",
  "description": "Desc...",
  "price": 423,
  "stock": 234,
  "category": "Graduation Cake",
  "image": "image url",
  "created_at": "2025-03-11T13:59:05.000Z"
}
``` 
  - [POST]   http://localhost:3000/products
---
>Request JSON Body:
``` 
{
  "name": "Product name",
  "description": "Desc...",
  "price": 423,
  "stock": 234,
  "category": "Graduation Cake",
  "image": "image url",
}
``` 


>Response JSON Body:
``` 
{
  message: "Product created"
  id: 2
}
``` 
  - [PATCH]  http://localhost:3000/products/:id
---
>Request JSON Body:
``` 
{
  "name": "Product name",
  "description": "Desc...",
  "price": 423,
  "stock": 234,
  "category": "Graduation Cake",
  "image": "image url",

``` 

>Response JSON Body:
```

``` 
  - [DELETE] http://localhost:3000/products/:id
---
>Request JSON Body:
`None`

>Response JSON Body:
``` 
{
  message: "Product deleted"
}
```
